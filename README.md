## Getting Started

Install the module with: `npm install holonomy`

```js
var holonomy = require('holonomy');
holonomy.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g holonomy
$ holonomy --help
$ holonomy --version
```
